# 🏋️ Training Booking Bot

Telegram-бот для записи на тренировки с учётом абонементов. Данные хранятся в Supabase, всё работает через кнопки — никакого свободного ввода.

---

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 📅 Запись | Выбор тренера → даты → времени. Автоматически проверяется остаток абонемента |
| ❌ Отмена | При отмене за 24+ часов тренировка возвращается в абонемент |
| 💪 Абонемент | В любой момент можно проверить, сколько тренировок осталось |
| 📢 Админ | При каждой записи или отмене бот присылает уведомление администратору |
| 🗄️ Supabase | Все данные сохраняются в облачной базе и не теряются при перезапуске |

---

## 🛠️ Технологии

- **Node.js** + **Telegraf** — Telegram Bot API
- **Supabase** (PostgreSQL) — база данных
- **JavaScript** (ES Modules)


## 🚀 Запуск проекта

### 1. Клонируйте репозиторий

git clone https://github.com/ваш_username/training-booking-bot.git
cd training-booking-bot

### 2. Установите зависимости

npm install

### 3. Создайте файл .env

BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_id
SUPABASE_URL=ваш_url
SUPABASE_ANON_KEY=ваш_ключ

### 4. Запустите бота

npm start

## 📬 Команды бота

/start - главное меню
/my - остаток тренировок

## 🗄️ Структура базы данных (Supabase)

users - пользователи и остаток тренировок
trainers - список тренеров
schedule - расписание (дата, время, количество мест)
bookings - записи клиентов
